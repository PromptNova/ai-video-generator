FAQ

Why did I get “405 Method Not Allowed” earlier?
That happens when the server route doesn’t accept POST on /upload. This frontend sends a POST with multipart/form-data.
Can I generate voice-over for hook #2–#5?
Yes. Generate once, then click Use on any hook. The app calls POST /apply-hook.
Where do the output files live?
The backend returns URLs like /static/outputs/<id>.mp4.
Can I host frontend separately from backend?
Yes. Add ?api=https://your-backend-domain to the URL.
